# React Assignments

React 练习项目合集，每个分支对应一次独立的练习。

## 技术栈

- React 19
- TypeScript
- Vite
- react-router v7

## 分支说明

| 日期 | 分支 | 内容 | 核心知识点 |
| --- | --- | --- | --- |
| 2026-04-27 | [react-router-assignment](../../tree/2026-04-27-react-router-assignment) | React Router 路由演示 | 二级路由、路由守卫、路由监听、独立路由配置、Token 鉴权 |
| 2026-05-18 | [effect-and-immutable-state](../../tree/2026-05-18-effect-and-immutable-state) | Hooks 与 state 更新 | useLayoutEffect vs useEffect、数组/对象的不可变更新 |

## 使用方式

```bash
# 克隆仓库
git clone https://github.com/Flysoft1337/react-assignments.git

# 切换到对应分支
git checkout 2026-05-18-effect-and-immutable-state

# 安装依赖并启动
npm install
npm run dev
```
